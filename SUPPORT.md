# Support

## How to Get Help

### Documentation

Start with the project documentation at **https://vignesh2027.github.io/detect-backend-threat**.

Key pages:
- [How It Works](https://vignesh2027.github.io/detect-backend-threat/guides/how-it-works/) — deep dives into every subsystem
- [Configuration Reference](https://vignesh2027.github.io/detect-backend-threat/reference/configuration/) — all environment variables
- [Development Guide](https://vignesh2027.github.io/detect-backend-threat/guides/development/) — local setup and adding new scanners
- [Deployment Guide](https://vignesh2027.github.io/detect-backend-threat/guides/deployment/) — production checklist

### GitHub Issues

For **bugs** and **feature requests**, open an issue at:
https://github.com/vignesh2027/detect-backend-threat/issues

Use the provided templates and include:
- Your OS and Go/Node.js version
- The exact command or API call that failed
- Relevant log output (redact any API keys)

### GitHub Discussions

For **questions**, **ideas**, and **general conversation**:
https://github.com/vignesh2027/detect-backend-threat/discussions

### Email

For **security vulnerabilities**, see [SECURITY.md](SECURITY.md).

For other private matters: **lkvarnesh@gmail.com**

## What Is Out of Scope

- Commercial support or SLAs (this is a community project)
- Support for EOL Go versions (< 1.22)
- Debugging issues caused by modifications to the core detection algorithms
